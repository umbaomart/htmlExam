# コムデ テスト用ウェブサイト

* * *

## 基本要件

/psd/pc.css（PC用）
/psd/sp.css（スマホ用）
のデザイン通りコーディングして下さい。

/html/ フォルダ内に index.html を作成し、CSSファイルは別ファイルで読み込んで下さい。
（CSSのディレクトリ階層、ファイル名の指定はありません。）

コーディングが完了したら /html/ フォルダを送って下さい。

* * *

## スタイル要件
- Retina対応
- レスポンシブ対応（メディアクエリを使用）
- ブレイクポイントは768px（769〜1000pxは見切れて良い）

* * *

## 対応ブラウザ

タブレットの対応は含まれません。下記のPC/スマホ端末に対応する実装を行って下さい。

### Windows
- Microsoft Edge

### Mac
- Google Chrome 最新版
- FireFox 最新版
- Safari 最新版

### iOS
- iOS 最新版 Safari

### Android
- Android 5.0〜 Google Chrome

* * *

## その他

- 使用していないコード、画像などのファイルは削除してください。
- Bootstrap などのテンプレートエンジンは使用せず、ピュアなCSSを使用して下さい。
- Sass などのプリプロセッサを使っても問題はありませんが、格納するファイルには非圧縮のファイルを格納して下さい。
- JavaScript が必要な箇所はないので、使用する必要はありません。

# COMMU:DE Trial Web Page

* * *

## Basic requirement

/psd/pc.css(for PC)
/psd/sp.css(for smartphone)
please coding as designed.

Create index.html in /html/ folder and load the CSS file as a separate file.
(The CSS directory hierarchy, file name specification are not specified.)

When coding is completed send /html/ folder.

* * *

## Style requirement
- Retina compatible
- Responsive (using media query)
- The break point is 768px (769 to 1000px can be overlooked)

* * *

## Supported browser

Tablet correspondence is not included.
Please implement corresponding to the following PC / smartphone.

### Windows
- Microsoft Edge

### Mac
- Google Chrome (Latest version)
- FireFox (Latest version)
- Safari (Latest version)

### iOS
- iOS (Latest version) Safari

### Android
- Android 5.0〜 Google Chrome

* * *

## Others

- Delete unused code, images and other files.
- Do not use template engine such as Bootstrap, please use pure CSS.
- There is no problem even if using a preprocessor such as Sass, but please store uncompressed file in the file to be stored.
- There is no need for JavaScript, so you do not need to use it.
